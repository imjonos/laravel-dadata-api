# Dadata Api Wrapper For Laravel



## Installation

Via Composer

``` bash
$ composer require codersstudio/dadataapi
```

## Usage

DadataApi::suggestCity("Москва", ["RU"]);
DadataApi::suggestCountry("Россия");
